# Changelog

## Version 0.7.0
- Added support for debugging:
  - Control commands:
    - Continue command
    - Pause command
    - Step-over command
    - Step-in command
    - Step-out command
    - Disconnect command

  - Features:
    - Set/Remove breakpoints
    - Set/Remove function breakpoints
    - Call stack display
    - Watch (evaluate expression)
    - Exception hint
    - Handle source receive from the engine
    - Sending source code from the vscode to the engine
